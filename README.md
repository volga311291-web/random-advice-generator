# 🎯 Генератор случайных советов

Простой и полезный веб-инструмент, который даёт мотивирующие и практические советы для улучшения жизни.

## 🌟 Возможности

- 🎲 Случайный совет по нажатию кнопки
- 📋 Копирование совета в буфер обмена
- ⌨️ Горячая клавиша **Пробел** для нового совета
- ✨ Красивая анимация при смене совета
- 📱 Адаптивный дизайн для мобильных устройств

## 🚀 Как использовать

1. Откройте [приложение](https://ваш_username.github.io/random-advice-generator)
2. Нажмите кнопку **"Получить совет"**
3. Полезный совет появится на экране
4. Нажмите **"Скопировать"**, чтобы сохранить совет

## 🛠️ Технологии

- HTML5
- CSS3 (градиенты, анимации, адаптивность)
- JavaScript (ES6+)
